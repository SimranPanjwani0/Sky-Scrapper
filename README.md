# Weather App
####  This Flutter weather application allows users to view the current weather of their city, <br> search for weather in other cities, and save cities to their favorites for quick access. <br>The app uses various packages to fetch weather data, manage permissions, and store favorite cities locally.


## Features
#### Current City Weather: Automatically fetches and displays the weather for the user's current location.
#### Search Weather: Search for the weather in any city worldwide.
#### Favorites: Add cities to a favorites list for quick access.
#### Local Storage: Saves favorite cities using a local SQLite database.
#### Permissions: Handles location permissions for fetching the user's current city.

## Usage
#### View Current Weather: On launching the app, it will request location permission and display the current weather for your city.
#### Search City Weather: Use the search feature to find weather information for any city.
#### Add to Favorites: Save frequently checked cities to your favorites list for quick access.
#### Manage Permissions: If location permission is not granted, the app will prompt the user to enable it.
#### Technologies Used
#### Flutter: Cross-platform framework for building the application.
#### Dart: Programming language used with Flutter.
#### State Management: Using Get package for state management.
#### Local Database: SQLite for storing favorite cities locally.


## Packages
#### List of Flutter packages used in this project:

#### http: For making API calls to fetch weather data.
#### logger: For logging information, errors, and debugging.
#### get: For state management and route management.
#### geolocator: For fetching the user's current location.
#### permission_handler: For managing location permissions.
#### sqflite: For local storage of favorite cities.
